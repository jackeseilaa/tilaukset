J Sailing — Absolut 37 Tilaukset (ILMAINEN, ei lisäappeja)
====================================================

Tämä paketti toimii iPadissa Safarilla "sovelluksena" (PWA) ja ilman maksullisia palveluita.
Julkaisu tehdään GitHub Pagesilla (0 €).

1) Julkaise GitHub Pagesiin (0 €)
---------------------------------
A) Tee GitHub-tili (jos ei ole)
B) Luo uusi repo: esim. "tilaukset" (Public)
C) Lataa NÄMÄ tiedostot repon juureen:
   - index.html
   - manifest.json
   - sw.js
   - icons/ (kansio ja sen 3 png:tä)

D) Repo → Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
   - Save

Saat osoitteen:
https://KAYTTAJANIMI.github.io/tilaukset/

2) Asenna iPadiin kotinäyttöön
------------------------------
Avaa osoite Safarissa → Jaa → Lisää Koti-näyttöön.

3) TALLENNUS / VARMUUSKOPIO
---------------------------
- Data tallentuu automaattisesti laitteeseen (offline).
- Tee varmuuskopio säännöllisesti:
  Dashboard → "Varmuuskopio (JSON)" ja tallenna iCloud Driveen.
- Palautus:
  Dashboard → "Tuo (JSON)"

4) VIENTI (CSV)
---------------
Dashboard → CSV Asiakkaat / CSV Purjehdukset / Kirjanpito CSV
(Tiedostot voi avata Excelissä; erotinmerkki on puolipiste ; )

Huom:
- "Tallenna tiedostoon" toimii vain, jos iPadOS/Safari tukee File Picker -tallennusta.
  Muuten käytä Varmuuskopio (JSON) -nappia.
