Kirjoita esim. "150.50" tai "1234.99" — nyt se ei häviä kesken!

Kuinka tämä korjaus toimii:

ENNEN (virhe):
input.addEventListener("input", (e) => {
  state.amount = e.target.value;
  render(); ← ONGELMA!
})

JÄLKEEN (korjaus):
input.addEventListener("input", (e) => {
  state.amount = e.target.value;
  // Ei render()! Vain päivitä state
})

Kun render() ei kutsu JOKAISEN näppäinpainalluksen jälkeen, input-kenttä pysyy vakaana ja käyttäjä voi kirjoittaa normaalisti.
